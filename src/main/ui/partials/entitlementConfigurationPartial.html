Custom Governance

This is the Entitlement Approval Configuration page. You can create detailed approval definitions for your application's entitlements
Default Configuration are the settings that will be applied to all entitlements that

  • are aggregated by a group aggregation task with the appropriate aggregation rule set
  • belong to an application that has not been defined here, yet

Defined Application both lists existing Application Definitions and lets you create a new Application Definition
To access the functions, click on the icon to open it.
After you finished defining your definitions, don't forget to save.
